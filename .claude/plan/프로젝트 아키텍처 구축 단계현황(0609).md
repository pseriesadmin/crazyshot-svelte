## 2. 즉시 처리할 사항 (UNBLOCK)

### UNBLOCK-1: SSR WebSocket Fix
**목표**: `npm run dev` 정상 실행  
**파일**: `src/hooks.server.ts`  
**수정 내용**: `require('ws')` → ESM `import ws from 'ws'` 방식으로 교체  
**접근법**:
```typescript
// hooks.server.ts 수정안
import ws from 'ws'
import type { Handle } from '@sveltejs/kit'

if (typeof globalThis.WebSocket === 'undefined') {
  ;(globalThis as unknown as Record<string, unknown>).WebSocket = ws
}

export const handle: Handle = async ({ event, resolve }) => resolve(event)
```
**검증**: `npm run dev` 실행 후 `/products` 접근 확인

**방법론**: TDD 강제 (sp2-tdd-agents 위임)  
**핵심 구현 항목**:

- 멱등성 키 중복 방지

**TDD 필수 케이스** (`payment.md` 참조):
- Happy: 정상 결제 → confirmed + payment 기록
- Edge: calc_at 30초 초과 → PRICE_EXPIRED, 금액 불일치 → 승인 거부
- Error: 결제 실패 → HOLD 해제, 동일 idempotency_key → 두 번째 무시

**핵심 파일 생성**:
```

```
